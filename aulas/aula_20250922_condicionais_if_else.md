## Objetivos

- Escrever decisões com if, elif, else.
- Encadeamento, aninhamento e boas práticas (early return, legibilidade).

---

## Exemplos

```python
nota = float(input("Nota: "))
if nota >= 7:
    print("Aprovado")
elif nota >= 5:
    print("Recuperação")
else:
    print("Reprovado")
```
